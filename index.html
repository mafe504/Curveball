Developer Updates

Version 3.0: The Magic Update

  • Wizard UI/UX Overhaul: Shifted from a static multi-column layout to a dynamic, focused 'Wizard' progression, paired with modern glassmorphism aesthetics and smooth animations.
  • AI Wizard Persona: Transformed the 'Boss System' into an engaging 'AI Wizard Assistant' that strictly aligns with the student's grade level using simple language, emojis, and clear bullet points.
  • Dynamic Visual Assets: The AI now generates embedded `[GRAPH]` and `[IMAGE]` tags, which the frontend intercepts to render live, animated telemetry bars and real-time crisis photography (via pollinations.ai).
  • Pure Challenge Engine: Strengthened AI logic to strictly present the catastrophic twist *without* offering any implicit hints or fixes, forcing true independent problem-solving.

Version 2.0: Systemic Foundation

  • Dynamic Vulnerability Engine: Replaced generic scenarios with context-aware AI constraints targeting specific material flaws.
  • Profile Calibration: Adjusts AI difficulty, vocabulary, and crisis complexity based on the student's level.
  • 3-Trial Pivot Loop: Requires strategic growth and forces users to address compounding complications.
  • Rigorous Constraints: Enforces a 15-word minimum constraint on student submissions.
